# Substrings

> B2.1.2 Construct programs that can extract and manipulate substrings. Altering, concatenating, or replacing substrings.

## What is a Substring?

A **substring** is a portion of a string—a sequence of characters extracted from a larger string. Think of it like cutting out a section of text from a sentence. For example, if you have the string `"Hello World"`, the substring `"World"` is just part of that original string.

Understanding how to work with substrings is essential for tasks like:
- Extracting usernames from email addresses
- Processing file extensions
- Validating input formats
- Text parsing and analysis

Substrings work with string [[B2.1.1_Variables|variables]], one of the fundamental data types in programming.

## Extracting Substrings

In Python, we can extract substrings using **slicing** notation: `string[start:end]`. This extracts characters from index `start` up to (but not including) index `end`.

```python
# Basic substring extraction
message = "Hello World"

# Extract "Hello" (characters 0-4)
greeting = message[0:5]
print(greeting)  # Output: Hello

# Extract "World" (characters 6-10)
subject = message[6:11]
print(subject)  # Output: World

# Omit start to begin from the beginning
first_five = message[:5]
print(first_five)  # Output: Hello

# Omit end to go to the end of the string
from_sixth = message[6:]
print(from_sixth)  # Output: World
```

### Negative Indexing

Python allows **negative indices** to count from the end of the string. This is particularly useful when you don't know the exact length.

```python
filename = "document.pdf"

# Extract the last 3 characters (file extension)
extension = filename[-3:]
print(extension)  # Output: pdf

# Extract everything except the last 4 characters
name_only = filename[:-4]
print(name_only)  # Output: document
```

## Manipulating Substrings

### Concatenating Strings

**Concatenation** means joining strings together. In Python, we use the `+` operator.

```python
# Concatenating strings
first_name = "Ada"
last_name = "Lovelace"

# Join with a space
full_name = first_name + " " + last_name
print(full_name)  # Output: Ada Lovelace

# Building a file path
folder = "documents"
filename = "report.txt"
path = folder + "/" + filename
print(path)  # Output: documents/report.txt
```

### Replacing Substrings

The `.replace()` method allows us to find and replace substrings within a string.

```python
# Basic replacement
sentence = "I love cats"
new_sentence = sentence.replace("cats", "dogs")
print(new_sentence)  # Output: I love dogs

# Replace all occurrences
text = "red red red"
updated = text.replace("red", "blue")
print(updated)  # Output: blue blue blue

# Replace with a limit (replace only first 2 occurrences)
text = "red red red"
updated = text.replace("red", "blue", 2)
print(updated)  # Output: blue blue red
```

### Altering Substrings

Since strings in Python are **immutable** (cannot be changed directly), we must create new strings when altering them. We can combine slicing with concatenation to modify specific parts.

```python
# Replace a portion of a string
original = "Hello World"

# Change "World" to "Python"
# Extract everything before "World", then concatenate "Python"
modified = original[:6] + "Python"
print(modified)  # Output: Hello Python

# Insert text in the middle
text = "I programming"
# Insert "love " after "I "
result = text[:2] + "love " + text[2:]
print(result)  # Output: I love programming
```

## Practical Examples

### Example 1: Email Validation

```python
# Extract username and domain from an email address
email = "student@school.edu"

# Find the position of the @ symbol
at_position = email.find("@")

# Extract username (everything before @)
username = email[:at_position]
print(f"Username: {username}")  # Output: Username: student

# Extract domain (everything after @)
domain = email[at_position + 1:]
print(f"Domain: {domain}")  # Output: Domain: school.edu
```

When processing user input like email addresses, you should use [[B2.1.3_Exceptions|exception handling]] to deal with invalid formats gracefully.

### Example 2: Processing a List of Filenames

```python
# Process multiple filenames to extract extensions
filenames = ["report.pdf", "image.jpg", "data.csv", "notes.txt"]

# Use a for loop to process each filename
for filename in filenames:
    # Find the position of the last dot
    dot_position = filename.rfind(".")
    
    # Extract the extension
    extension = filename[dot_position + 1:]
    
    # Extract the name without extension
    name = filename[:dot_position]
    
    print(f"File: {name}, Type: {extension}")

# Output:
# File: report, Type: pdf
# File: image, Type: jpg
# File: data, Type: csv
# File: notes, Type: txt
```

### Example 3: Censoring Sensitive Information

```python
# Replace sensitive information in a string
message = "My credit card number is 1234-5678-9012-3456"

# Find and replace the card number
censored = message.replace("1234-5678-9012-3456", "****-****-****-****")
print(censored)  # Output: My credit card number is ****-****-****-****

# Alternative: Replace any sequence of digits
# This example shows manual substring manipulation
original = "Account: 987654321"
account_start = original.find(": ") + 2
account_number = original[account_start:]

# Create censored version
censored_account = "*" * len(account_number)
result = original[:account_start] + censored_account
print(result)  # Output: Account: *********
```

### Example 4: Text Formatting

```python
# Convert a list of names to proper case
names = ["ALICE SMITH", "bob JONES", "charlie BROWN"]

# Process each name using a for loop
formatted_names = []

for name in names:
    # Convert to lowercase first
    lowercase_name = name.lower()
    
    # Capitalise first letter of each word
    formatted = lowercase_name.title()
    
    formatted_names.append(formatted)

# Display results
for name in formatted_names:
    print(name)

# Output:
# Alice Smith
# Bob Jones
# Charlie Brown
```

## Common String Methods for Substring Operations

Python provides several built-in methods that work with substrings:

| Method | Description | Example |
|--------|-------------|---------|
| `.find(substring)` | Returns the index of the first occurrence, or -1 if not found | `"hello".find("ll")` → `2` |
| `.rfind(substring)` | Returns the index of the last occurrence | `"hello".rfind("l")` → `3` |
| `.replace(old, new)` | Replaces all occurrences of old with new | `"hi hi".replace("hi", "bye")` → `"bye bye"` |
| `.split(delimiter)` | Splits string into a list of substrings | `"a,b,c".split(",")` → `["a", "b", "c"]` |
| `.join(list)` | Joins list elements into a string | `"-".join(["a", "b"])` → `"a-b"` |
| `.strip()` | Removes whitespace from both ends | `" hi ".strip()` → `"hi"` |
| `.upper()` | Converts to uppercase | `"hello".upper()` → `"HELLO"` |
| `.lower()` | Converts to lowercase | `"HELLO".lower()` → `"hello"` |

## Key Takeaways

1. **Substrings** are portions of strings extracted using slicing notation `[start:end]`
2. **Concatenation** joins strings together using the `+` operator
3. **Replacement** uses the `.replace()` method to substitute substrings
4. Strings are **immutable**, so alterations create new strings
5. Use **for loops** to process multiple strings efficiently
6. Python provides many built-in methods for common substring operations
